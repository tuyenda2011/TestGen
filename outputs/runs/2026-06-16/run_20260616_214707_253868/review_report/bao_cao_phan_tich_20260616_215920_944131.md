- **Cách sửa:**  
  - Thêm một test **package‑private** (hoặc sử dụng reflection) để gọi trực tiếp `checkActive()` và xác nhận `IllegalStateException` với thông điệp `"Account is inactive"`.  
  - Ví dụ:  
    ```java
    @Test
    @DisplayName("TC-004a: checkActive throws IllegalStateException when account is inactive")
    void checkActive_inactiveAccount_throwsException() throws Exception {
        BankAccount account = new BankAccount("ACC123", 0.0);
        // deactivate via reflection vì method private
        java.lang.reflect.Method m = BankAccount.class.getDeclaredMethod("checkActive");
        m.setAccessible(true);
        account.deactivate(); // make inactive
        IllegalStateException ex = assertThrows(IllegalStateException.class, () -> m.invoke(account));
        assertEquals("Account is inactive", ex.getMessage());
    }
    ```
