## Rủi ro maintainability
- **Sử dụng `time.sleep(0.2)`** để chờ UI cập nhật có thể gây flaky trên máy chậm hoặc khi network latency tăng. Nên dùng explicit wait (WebDriverWait) để chờ element `#total` có giá trị thay đổi.  
- **Hard‑code selector strings** trong hàm `get_output` (`By.CSS_SELECTOR, selector`) nhưng các test truyền selector dạng `"#total"`; nếu thay đổi cấu trúc DOM (ví dụ đổi `output` thành `div`), tất cả test sẽ phá vỡ mà không có thông báo rõ ràng.  
- **Khởi tạo Edge driver** yêu cầu môi trường có Edge và driver tương thích; nếu môi trường CI không cài Edge, các test sẽ không chạy. Nên cân nhắc dùng Chrome/Chromium hoặc WebDriverManager để tự động tải driver.

   ```python
   from selenium.webdriver.support.ui import WebDriverWait
   from selenium.webdriver.support import expected_conditions as EC

       driver.find_element(By.ID, "calculate").click()
       WebDriverWait(driver, 5).until(

   ```
