## Rủi ro maintainability
- **Phụ thuộc vào trình duyệt Edge**: Nếu môi trường không cài EdgeDriver hoặc phiên bản không tương thích, các test sẽ thất bại (flaky). Nên cân nhắc dùng Chrome/Firefox hoặc WebDriverManager để tự động tải driver.
- **Dữ liệu tĩnh trong HTML**: Khi thay đổi nội dung HTML (ví dụ thay đổi `id` hoặc `value` mặc định) các test có thể bị break mà không có thông báo rõ ràng.

## Gợi ý sửa ngay
1. **Thêm test cho các kịch bản negative**  
   ```python
   @pytest.mark.parametrize(
       "subtotal,weight,destination,tier,fragile,credit,expected_error",
       [
           ("abc", 10, "domestic", "standard", False, 0, "Invalid input"),
           (120, None, "domestic", "standard", False, 0, "Invalid input"),
       ],
   )
   def test_invalid_inputs(driver, subtotal, weight, destination, tier, fragile, credit, expected_error):
       # Điền dữ liệu (đối với None hoặc non‑numeric, dùng chuỗi rỗng)
       fill_form(driver,
                 subtotal if isinstance(subtotal, (int, float)) else "",
                 weight if isinstance(weight, (int, float)) else "",
                 destination, tier, fragile, credit)
       # Kiểm tra thông báo lỗi
       error_text = driver.find_element(By.ID, "error").text
       assert expected_error in error_text
       # Tổng không thay đổi (giữ giá trị ban đầu)
       total_text = driver.find_element(By.ID, "total").text
       assert total_text == "0.00"   # hoặc giá trị trước đó nếu có
   ```
   - Đảm bảo `fill_form` có thể nhận `""` cho các trường không hợp lệ.
   - Kiểm tra rằng `error` không rỗng và `total` không thay đổi.

2. **Cải thiện việc khởi tạo driver**  
   - Sử dụng `webdriver_manager` để tự động tải driver, giảm phụ thuộc môi trường:
     ```python
     from webdriver_manager.microsoft import EdgeChromiumDriverManager
     driver = webdriver.Edge(EdgeOptions(), executable_path=EdgeChromiumDriverManager().install())
     ```
